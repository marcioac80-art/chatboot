# chatboot
projeto de envio de mensagem automática para técnicos do brasil todo.
projeto criado devido uma demanda da empresa, para saber se controlar a troca e reincidencia de uma peça de uma maneira simples e funcional.
Esse projeto e bem simples foi criado um link e enviado para todos os tecnicos do brasil , ao preencher os dados e enviado para uma plataforma da empresa e geral um dashborad automatico.
